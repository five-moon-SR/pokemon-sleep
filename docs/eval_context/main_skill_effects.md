# メインスキル効果値まとめ（重要部分のみ）

各スキルの**効果値テーブルと、その解釈に必須な情報だけ**を集約。仕様・考察・運用論は別ファイル `main_skill_notes.md` を参照。

データソース: Wikiから手動転記（Ver.3.2.0 前後）。

## 共通の解釈ルール
- **ランダム版の期待値**: 一様分布仮定で `(min+max)/2`。実測上 Lv7まで `固定×1.25`、Lv8で `固定×1.15`（ゆめかけらゲットSのみLv8あり）。
- **フィールドボーナス**: エナジーチャージ系のみ効果量に加算される（ステータス画面表示はボーナス前）。
- **げんき回復上限**: 150%（オール/エール/チャージ全て、超過分切り捨て）。
- **派生スキル**: 親と同じ効果値テーブルを使う前提だが、**ナイトメア・マイナス・きのみジュース・みかづきのいのり・ばけのかわ等は親と異なる可能性あり（未調査）**。実装側でフラグを立てる余地を残すこと。

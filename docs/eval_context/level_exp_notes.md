# レベル / EXP / アメ仕様（AI読み取り用ノート）

## ソース
- `貼り付けデータ集/基本データ.txt`（PukiWiki形式、Wiki Ver.2.10.0時点）

## レベルアップの恩恵
1. おてつだい時間の短縮（Lv.1あたり0.2%、Lv.2で-6s）
2. きのみのエナジー増加（→ `utils/berry_energy.py` 参照）
3. サブスキル解放（Lv.10/25/50/75/100、上限65のため75/100は現状解放不可）
4. 食材枠解放（第2枠=Lv.30、第3枠=Lv.60）
5. 進化解禁（Lvに依存しない進化もあり）

## レベル上限
- Ver.2.7.0時点でリサーチランク上限65、ポケモンLv上限も**65**
- 上限到達後の余剰EXPは「レベル上限ボーナス」に変換
- リサーチランクとLv上限が同時到達した場合は余剰EXPは捨てられる

## 経験値タイプ
- ほとんどのポケモンは **600タイプ**
- 一部 **900タイプ**（必要EXPが600の **1.5倍**）
- ゆめのかけらの必要量はどちらも同じ

## 睡眠リサーチでのEXP獲得
```
睡眠EXP = 睡眠スコア × 性格補正 × 睡眠EXPボーナス補正 × せいちょうのおこう補正 × イベント補正
```
- 睡眠スコア最大100＝補正なしで100EXP/日（アメ4個分）
- 性格補正: ▲1.18 / ▼0.82
- 睡眠EXPボーナス（サブスキル）: 1+(0.14 × ボーナス数)
- せいちょうのおこう: ×2
- グッドスリープデー: ×2（満月前後）/ ×3（満月当日）

理論最大例（Lv30以上、編成5体全員睡眠EXPボーナス、おこう＋満月）:
`100 × 1.18 × 1.7 × 2 × 3 = 1203.6 EXP/日 ＝ アメ48個分`

## アメ入手方法
1. 睡眠リサーチで寝顔を撮影
2. ソーシャルリサーチでフレンドから受け取る
3. おてつだいポケモンが拾う
4. ポケモンを博士に送る（逃がす）
5. ばんのうアメを使用する

## 派生メモ（実装での使い所）
- Lv.30/Lv.60 で食材枠が増える → 評価器で「将来性モード」のスロット解放判定に使用済（owned.py）
- アメ・かけら累計表は将来「育成プランナー」機能でLv目標までの所要量試算に使える
- 性格補正▲▼の倍率は `data/nature.json` の `_meta.modifiers` と一致（exp 軸：1.18/0.82）

## 食材構成（AAA / AAB / ABB / ABC …）— 2026-07-23

### 定義
- 食材スロットは Lv1 / Lv30 / Lv60 で解放。**スロット1 = A枠確定、スロット2 = A or B、スロット3 = A or B or C** のランダム抽選
- 個体の「3スロットが何枠を引いたか」を並べた表記が食材構成。例: AAA(全部A) / AAB / ABA / ABB / AAC / ABC
- 種族ごとに a/b/c 枠の食材と個数は固定（pokemon_master.json の ingredients。qty は**スロット位置順**）

### 厳選の定石（Game8 / ポケらく / note等の共通見解）
- **食材得意 → AAA が基本理想**。狙い食材のピンポイント大量供給。「食材ごとの専門家」を揃える思想。弱点はレシピ環境次第で腐ること
- **ABB** = 2種バランス供給で編成柔軟性が高い次善。**AAB** も同系
- **ABC** は器用貧乏になりがち。A/B/C 全部が優秀食材の種族（カイリュー等）でのみ肯定論あり
- **スキル得意** は構成より「拾う個数が少ない食材」が◎（所持数圧迫を避ける）
- **きのみ得意** は構成不問

### ツールとの対応
- 個体の構成は `ingredient_1/2/3`（食材名）から `utils/food_expectation.find_food_origin` で枠(a/b/c)に逆引きして導出できる → `composition_string()`
- 期待値計算（`qty_at_slot`）はスロット位置ベースなので構成の影響は織り込み済み
- 未解放/未入力スロットは「?」表記（このツールは先行入力可なので、入力済みなら現在Lvに関わらず確定表示）

## コミュニティティア表（最強ランキング）— 2026-07-23

### 定義
- 攻略サイトが計算/検証ベースで付けるポケモンの強さ格付け（S/A/B/C…）。食材軸・総エナジー軸など観点別
- 採用ソース: **RaenonX ポケモンティア表(食材軸)** https://pks.raenonx.cc/ja/pokedex/tier （計算ベースで更新頻度が高い）

### 捕獲方針への使い方
- **ティア上位ほど理想構成で確保する価値が高い**: 食材得意=AAA、きのみ得意=構成不問、スキル得意=低食材
- ツールでは data/community_tier.json に保存し、`utils/community_tier.py` 経由で:
  - 捕獲優先度スコアにティア係数（S=2.0 / A=1.6 / B=1.3 / C=1.1 / 未掲載=1.0）を乗算
  - 食材・育成戦略ページ「強ポケ捕獲方針」で 未所持→捕獲候補 / 所持だが構成違い→引き直し候補 を判定
- 更新運用: RaenonXの表示仕様上 S/A/Bは全量・Cは上位10件のみ取得可。Playwrightで再取得して JSON を差し替える
